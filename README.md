# 🛡️ Cloud-SIEM-Log-Automation

본 프로젝트는 인프라 및 클라우드 환경에서 발생하는 보안 로그(Auth Log)를 실시간으로 분석하여, 무단 접근 시도(Brute Force Attack) 및 이상 징후를 탐지·시각화하는 자동화 SIEM(Security Information and Event Management) 시스템입니다.

단순 횟수 기반 탐지의 한계를 극복하기 위해 **시간 윈도우(Time Window)** 알고리즘을 적용하여 오탐을 최소화하고 실효성 있는 위협 인텔리전스를 제공합니다.

## 🏗️ SIEM 데이터 탐지 흐름 (Data Flow)

```text
[ 원시 보안 로그 인입 ] (server_auth.log)
           │
           ▼
┌────────────────────────────────────────────────────────┐
│ 1. 정규식 기반 로그 파싱 레이어                        │
│    - 로그 내 타임스탬프 및 소스 IP 패턴 자동 추출      │
└──────────────────────────┬─────────────────────────────┘
                           │
                           ▼
┌────────────────────────────────────────────────────────┐
│ 2. IP별 타임라인 정렬 및 분석                          │
│    - IP 대역별 로그인 실패 시간대 시계열 정렬          │
└──────────────────────────┬─────────────────────────────┘
                           │
                           ▼
┌────────────────────────────────────────────────────────┐
│ 3. 시간 윈도우(Time Window) 검증 엔진                  │
│    - 임계치 설정: 5분 이내 연속 3회 이상 실패 시       │
│    - Brute Force 공격 시나리오 정밀 매칭               │
└──────────────────────────┬─────────────────────────────┘
                           │ (위협 탐지 시)
                           ▼
┌────────────────────────────────────────────────────────┐
│ 4. 위협 분석 경보 및 인텔리전스 제공                   │
│    - 실시간 ALERT 경보 및 공격 타임프레임 분석 출력    │
└────────────────────────────────────────────────────────┘
```

## 🛠️ 핵심 기능 및 기술 스택
* **시간 기반 이상 징후 탐지:** 5분(Time Window) 이내 특정 IP로부터 누적되는 실패 패턴을 정밀 추적하여 실제 침해 시도만 필터링.
* **정규 표현식(Regex) 파싱:** 파이썬 `re` 모듈을 통한 대용량 로그 데이터 파싱 자동화.
* **형상 관리:** Git Branch 전략 및 `git stash`를 활용한 체계적인 기능 고도화 파이프라인 수립.

## 🚀 프로젝트 실행 방법
1. 로컬 환경에서 아래 명령어를 통해 탐지 엔진을 실행합니다:
```bash
python3 siem_parser.py
```
